## 概要

フロントエンド開発は、ウェブサイトやアプリケーションのユーザーインターフェイスを作成するプロセスです。この記事では、HTML、CSS、およびJavaScriptの基本に焦点を当てます。

### HTMLの基礎

- **HTML（HyperText Markup Language）**は、ウェブページの構造を定義するために使用されます。
- 基本的な構造要素:
  ```html
  <!doctype html>
  <html>
    <head>
      <title>ページタイトル</title>
    </head>
    <body>
      <h1>これは見出し</h1>
      <p>これは段落です。</p>
    </body>
  </html>
  ```

### CSSの基礎

- **CSS（Cascading Style Sheets）**は、ウェブページのデザインとレイアウトを制御します。
- 基本的なスタイルの例:

  ```css
  body {
    font-family: Arial, sans-serif;
  }

  h1 {
    color: navy;
  }
  ```

### JavaScriptの基礎

- **JavaScript**は、ウェブページに動的要素やインタラクティビティを追加するために使用されます。
- 簡単なスクリプト例:
  ```javascript
  document.getElementById('demo').innerHTML = 'こんにちは、JavaScript!';
  ```
